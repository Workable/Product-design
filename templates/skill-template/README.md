# skill-template

A starter for new Claude skills. Copy this folder, rename it, and edit `SKILL.md`.

```bash
cp -r templates/skill-template skills/your-skill-name
```

Then edit `skills/your-skill-name/SKILL.md` — replace the placeholder `name` and `description` in the frontmatter, then write the body.

See `docs/adding-a-skill.md` for full guidance.
