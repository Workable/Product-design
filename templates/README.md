# templates/

Reusable starting points for design work.

## What goes here

- Design doc templates (research briefs, design specs, review docs)
- Presentation templates (`.pptx`, links to Figma decks)
- Process templates (handoff checklists, QA checklists)
- The `skill-template/` — the canonical starting point for new Claude skills

## Adding a template

```
templates/
  your-template-name/
    README.md         # what it is, when to use, who maintains it
    template-file.*   # the template itself
```
